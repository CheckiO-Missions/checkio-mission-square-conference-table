Vous devez combiner plusieurs bureaux longs pour obtenir une table de conférence carrée de la taille spécifiée.

  • Les valeurs d'entrée sont les largeurs des bureaux longs (tuples d'entiers) et la longueur du côté de la table (entier). Les bureaux longs peuvent avoir n'importe quelle largeur et leur profondeur est toujours de 1.
  • La réponse est la combinaison des pupitres longs qui forment les quatre côtés de la table. Il doit s'agir d'une liste de 4 tuples, chaque tuple étant un ou plusieurs entiers. Les tuples adjacents de la liste résultante représentent en fait les côtés adjacents de la table.

REMARQUE:

  • Étant donné qu'il peut y avoir plus d'une réponse correcte, vous pouvez utiliser un vérificateur.
  • Tous les tests ont des réponses correctes.
  • Il n'est pas toujours nécessaire d'utiliser tous les pupitres longs.

Entrée: Les pupitres longs (un tuple d'entiers) et la longueur du côté de la table (un entier).

Sortie: Les combinaisons des pupitres longs (une liste de 4 tuples d'entiers).

square_conference_table((1, 2, 2, 3, 4, 4), 4):
ex_1: ex_2:
[(4,), (3,), (2,1), (2,)] [(4,), (2,), (4,), (2,)]

Condition préalable:

  • len(long_desks) <= 20
  • longueur_côté <= 60

Exemples: 

from itertools import chain
from collections import Counter
from typing import Callable


def checker(fn: Callable, ds: tuple[int, ...], s: int) -> bool:
    result = fn(ds, s)
    if Counter(chain(*result)) <= Counter(ds):
        difs = [s - sum(r) for r in result]
        return difs in ([1, 1, 1, 1], [2, 0, 2, 0], [0, 2, 0, 2]) or sorted(difs) == [0, 1, 1, 2]
    return False
