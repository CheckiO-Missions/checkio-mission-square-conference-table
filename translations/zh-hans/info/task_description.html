您必须将几张长桌组合起来，才能制作出指定尺寸的方形会议桌。

  • 输入值是长桌的宽度（整数元组）和桌子的边长（整数）。 长桌的宽度不限，深度始终为 1。
  • 答案是组成桌子四边的长桌组合。 它必须是一个包含 4 个图元的列表，每个图元是一个或多个整数。 结果列表中的相邻元组实际上代表桌子的相邻边。

注意

  • 由于正确答案可能不止一个，因此使用校验器时。
  • 所有测试都有正确答案。
  • 不一定非要用完所有的长课桌。

输入长课桌（整数元组）和桌子边长（整数）。

输出：长桌的组合（由 4 个整数元组组成的列表）。

square_conference_table((1, 2, 2, 3, 4, 4), 4) ：
ex_1： ex_2：
[(4,), (3,), (2,1), (2,)] [(4,), (2,), (4,), (2,)]

先决条件：

  • len(long_desks) <= 20
  • 边长 <= 60

示例 

from itertools import chain
from collections import Counter
from typing import Callable


def checker(fn: Callable, ds: tuple[int, ...], s: int) -> bool:
    result = fn(ds, s)
    if Counter(chain(*result)) <= Counter(ds):
        difs = [s - sum(r) for r in result]
        return difs in ([1, 1, 1, 1], [2, 0, 2, 0], [0, 2, 0, 2]) or sorted(difs) == [0, 1, 1, 2]
    return False
