Sie müssen mehrere lange Tische kombinieren, um einen quadratischen Konferenztisch in der angegebenen Größe zu erhalten.

  • Die Eingabewerte sind die Breiten der langen Tische (Tupel aus ganzen Zahlen) und die Seitenlänge des Tisches (ganze Zahlen). Die langen Tische sind beliebig breit und ihre Tiefe ist immer 1.
  • Die Antwort ist die Kombination der langen Pulte, die die vier Seiten des Tisches bilden. Es muss sich um eine Liste von 4 Tupeln handeln, wobei jedes Tupel eine oder mehrere ganze Zahlen ist. Die benachbarten Tupel in der resultierenden Liste müssen tatsächlich benachbarte Seiten des Tisches darstellen.

HINWEIS:

  • Da es mehr als eine richtige Antwort geben kann, verwenden Sie einen Checker.
  • Alle Tests haben richtige Antworten.
  • Sie müssen nicht immer alle langen Tische ausnutzen.

Eingabe: Die langen Tische (ein Tupel von ganzen Zahlen) und die Seitenlänge des Tisches (eine ganze Zahl).

Ausgabe: Die Kombinationen der langen Tische (eine Liste von 4 ganzzahligen Tupeln).

from itertools import chain
from collections import Counter
from typing import Callable


def checker(fn: Callable, ds: tuple[int, ...], s: int) -> bool:
    result = fn(ds, s)
    if Counter(chain(*result)) <= Counter(ds):
        difs = [s - sum(r) for r in result]
        return difs in ([1, 1, 1, 1], [2, 0, 2, 0], [0, 2, 0, 2]) or sorted(difs) == [0, 1, 1, 2]
    return False
