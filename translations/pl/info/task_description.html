Muszą Państwo połączyć kilka długich biurek, aby stworzyć kwadratowy stół konferencyjny o określonym rozmiarze.

  • Wartościami wejściowymi są szerokości długich biurek (krotki liczb całkowitych) i długość boku stołu (liczba całkowita). Długie biurka mogą mieć dowolną szerokość, a ich głębokość zawsze wynosi 1.
  • Odpowiedzią są kombinacje długich ławek, które tworzą cztery boki stołu. Musi to być lista 4 krotek, z których każda jest co najmniej jedną liczbą całkowitą. Sąsiednie krotki na wynikowej liście muszą faktycznie reprezentować sąsiednie boki stołu.

PROSZĘ ZAUWAŻYĆ:

  • Ponieważ może istnieć więcej niż jedna poprawna odpowiedź, proszę użyć sprawdzarki.
  • Wszystkie testy mają poprawne odpowiedzi.
  • Nie zawsze trzeba wykorzystywać wszystkie długie biurka.

Dane wejściowe: Długie biurka (krotka liczb całkowitych) i długość boku stołu (liczba całkowita).

Dane wyjściowe: Kombinacje długich biurek (lista 4 krotek liczb całkowitych).

from itertools import chain
from collections import Counter
from typing import Callable


def checker(fn: Callable, ds: tuple[int, ...], s: int) -> bool:
    result = fn(ds, s)
    if Counter(chain(*result)) <= Counter(ds):
        difs = [s - sum(r) for r in result]
        return difs in ([1, 1, 1, 1], [2, 0, 2, 0], [0, 2, 0, 2]) or sorted(difs) == [0, 1, 1, 2]
    return False
